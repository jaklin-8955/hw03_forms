Используемые технологии

  • Python 3.9+
  • Django 4.2
  • SQLite3 (база данных)
  • HTML5 / CSS3 / Bootstrap 5
  • Git (система контроля версий)

Технологический стек проекта

Проект Yatube построен на мощном и гибком фреймворке Django. Для фронтенда используется Bootstrap 5, что позволяет быстро создавать адаптивный и приятный дизайн.

В ходе разработки я углубил знания по следующим темам:

  • Маршрутизация URL и пространства имён (namespaces).
  • Работа с формами и CBV (CreateView, TemplateView).
  • Система авторизации и регистрации пользователей (django.contrib.auth).
  • Отправка писем с восстановлением пароля через файловый бэкенд.
  • Использование статических файлов и медиа.

Пока не всё получается идеально, но я уверенно двигаюсь вперёд. Планирую добавить комментарии, пагинацию и возможность загружать изображения.

Спасибо, что заглянули на страницу «Технологии»!
